# requirements

## 1. 目的

禁煙アプリとして、記録・データ管理に加え、期間別の傾向分析を提供する。

## 2. 分析タブ要件

### R-A01 基本

- Analysisタブでグラフ分析を提供する
- 分析モードは `時間 / 週 / 月`
- 対象期間を前後に切り替え可能（週単位または月単位）

### R-A02 週分析

- 対象週の月〜日の本数を表示
- 同時に過去4週間平均（曜日別）を表示
- 対象週と平均の差が比較できること

### R-A03 月分析

- 対象月の日別本数を表示
- 同時に過去4か月平均（日別）を表示
- グラフとは別領域に対象月の金額を表示

### R-A04 時間分析

- `全日 / 各曜日` を選択可能
- 選択条件に対して時間帯別（0〜23時）本数を折れ線表示
- 対象期間は1週間
- 過去4週間平均の線も重ねる
- 対象週の前後切り替え可能

### R-A05 分析品質補正（仕様修正）

- 一括登録（`trendEligible=false`）は時刻の信頼性が低いため、時間分析から除外する
- 週・月分析は日単位比較のため、一括登録も集計に含める

## 3. Dataタブ要件（継続）

- Dataタブで追加・一括追加・変更・削除を提供
- カレンダーは月表示、当日デフォルト選択

## 4. Homeタブ要件（継続）

- 本日の本数、経過時間、今月金額、履歴、トレンド矢印、平均値を表示
