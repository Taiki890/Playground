# 分析タブ 手動テスト仕様書

## 対象データ
- ファイル: `docs/test-data/analysis-import-manual.json`
- 1箱あたり金額 (`packPrice`): 620円
- 3月合計本数: 32本
- 3月の金額期待値: `floor(32 * 620 / 20) = 992円`

## 事前準備
1. 設定タブで `Import` から `analysis-import-manual.json` を読み込む。
2. 分析タブへ移動する。

## テストケース

### TC-01: 月表示(2026年3月)の日別本数と金額
1. 分析モードを `月` に切り替える。
2. 期間表示が `2026/03` になるまで前後ボタンで移動する。
3. グラフの日別棒を確認する。
4. 月サマリーの金額を確認する。

期待結果:
- 3/1 が `10本`
- 3/9 が `3本`
- 3/11 が `3本`
- 3/14 が `4本`
- 3/15 が `5本`
- 3/20 が `7本`
- 上記以外の日は `0本`
- 月サマリー金額が `992円`

### TC-02: 週表示(2026-03-09週)の曜日別本数
1. 分析モードを `週` に切り替える。
2. 期間表示が `03/09 - 03/15` になるように移動する。
3. 曜日別の棒グラフを確認する。

期待結果:
- 月曜: `3本` (2本 + 1本)
- 火曜: `0本`
- 水曜: `3本`
- 木曜: `0本`
- 金曜: `0本`
- 土曜: `4本`
- 日曜: `5本`

### TC-03: 時間表示(同週・曜日フィルタなし)
1. 分析モードを `時間` に切り替える。
2. 期間表示を `03/09 - 03/15` に合わせる。
3. 曜日フィルタを `全て` にする。
4. 時間別線グラフを確認する。

期待結果:
- 08時: `2本` (08:40の1本は `trendEligible=false` のため除外)
- 09時: `4本`
- 11時: `0本` (3/20のデータは対象週外)
- 22時: `3本`
- その他の時間帯は `0本`

### TC-04: 時間表示(同週・月曜フィルタ)
1. TC-03 の状態で曜日フィルタを `月` に変更する。
2. 時間別線グラフを確認する。

期待結果:
- 08時: `2本`
- それ以外: `0本`

## 補足
- 時間表示は `trendEligible=true` のレコードのみ集計対象。
- 週表示と月表示は `trendEligible` に関係なく集計対象。
- 2月・4月データが含まれているため、月切替時に3月集計へ混入していないことも確認できる。
