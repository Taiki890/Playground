# acceptance criteria

## AC-01 ナビ表示

- 下部タブに `Home / Analysis / Data / Settings` が表示される
- `Data` が遷移可能である

## AC-02 Data初期表示

- Dataタブ初回表示で当日が選択される
- 月表示カレンダーが表示される
- 選択日ラベルと一覧エリアが表示される

## AC-03 カレンダー操作

- 前月/次月ボタンで月移動できる
- 日付選択で一覧が選択日に切り替わる

## AC-04 一覧表示

- 選択日のデータが時刻、本数、分析利用フラグ付きで表示される
- データがない日は空メッセージを表示する

## AC-05 追加

- `追加` で選択日に1件追加できる
- 追加データは `quantity=1`, `trendEligible=true`

## AC-06 一括追加

- `一括追加` で選択日に本数指定追加できる
- 追加データは `quantity>=2`, `trendEligible=false`

## AC-07 変更/削除

- 一覧行から `変更` で時刻・本数・分析利用フラグを更新できる
- 一覧行から `削除` で1件削除できる

## AC-08 集計連動

- Dataタブでの追加/変更/削除がHomeの日次本数・今月金額に反映される
- `trendEligible=false` のデータはトレンド計算に含まれない
