# acceptance criteria

## AC-A01 タブ

- Analysisタブが遷移可能
- Dataタブ名がDataで表示される

## AC-A02 モード切替

- `時間 / 週 / 月` で表示が切り替わる

## AC-A03 週分析

- 対象週の日別本数が表示される
- 過去4週間平均が同一グラフに表示される

## AC-A04 月分析

- 対象月の日別本数が表示される
- 過去4か月平均が同一グラフに表示される
- 月金額がグラフ下に表示される

## AC-A05 時間分析

- 全日/曜日選択で時間別グラフが更新される
- 対象週と過去4週間平均が表示される
- 一括登録データ（trendEligible=false）が時間分析から除外される

## AC-A06 期間移動

- 前/次ボタンで週または月を移動できる
- 表示ラベルが更新される

## AC-A07 Data整合

- Data追加/編集/削除後にAnalysisとHomeに反映される
