# acceptance criteria

## AC-01 Home基本表示

- 本日の本数、トレンド矢印、平均値、経過時間、今月金額、履歴が表示される

## AC-02 記録登録

- `+` 押下で `quantity=1` のレコードが追加される
- 当日本数、履歴、今月金額が更新される

## AC-03 経過時間

- 最新記録からの経過時間が `hh:mm:ss` で表示される
- 1秒ごとに表示が進む

## AC-04 トレンド

- トレンド矢印が `↑ / → / ↓` のいずれかで表示される
- 矢印の横に平均値が `avg N.N` 形式で表示される
- `trendEligible=false` のデータは判定に含まれない

## AC-05 履歴

- 履歴は降順表示
- 最大10件表示
- 履歴領域のみスクロール可能

## AC-06 Settings

- packPriceを保存できる
- JSONエクスポートできる
- JSONインポートで復元できる

## AC-07 UI操作性

- Homeは1画面内で利用できる
- `+` ボタンは横長で押しやすい

## AC-08 起動耐性

- `service worker` 登録失敗時も画面初期化は継続する
